# Privacy Policy — Tabs to PDF

Last updated: 2026-05-23

## Data Collection
This extension does not collect, store, or transmit any personal data or user information.

## Debugger API Usage
The Chrome Debugger API is used solely to invoke the browser's built-in 
`Page.printToPDF` function to generate PDF files from open tabs.  
No page content, URLs, browsing history, or user data is sent to any external server.

## Local Processing
All PDF generation happens entirely within your local browser.  
No data leaves your device.
