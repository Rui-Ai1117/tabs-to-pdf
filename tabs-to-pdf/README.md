# Tabs to PDF — Chrome 拡張機能

複数タブを一括でPDF化できるChrome拡張機能です。

## 機能

- **全タブ一括変換** — 現在のウィンドウの全タブをPDF化
- **タブ選択** — 変換したいタブだけチェックして変換
- **現在のタブのみ** — アクティブなタブのみを変換
- **個別 or 結合** — タブごとに個別PDF / まとめて連番PDF
- **用紙設定** — A4 / Letter / A3、縦横対応
- **余白設定** — デフォルト / なし / 最小
- **背景印刷** — 背景色・画像の印刷ON/OFF
- **タブ間ウェイト** — サーバー負荷を抑える待機時間設定

## インストール方法

1. Chromeのアドレスバーに `chrome://extensions` を入力してEnter
2. 右上の **「デベロッパーモード」をON** にする
3. **「パッケージ化されていない拡張機能を読み込む」** をクリック
4. この `tabs-to-pdf` フォルダを選択する
5. 拡張機能のアイコンがツールバーに表示される ✅

## 使い方

1. ツールバーのアイコンをクリック
2. モードを選択（全タブ / 選択 / 現在のタブ）
3. 必要に応じて設定を調整
4. 「PDF に変換」ボタンをクリック
5. ダウンロードフォルダの `TabsToPDF/` フォルダにPDFが保存される

## 注意事項

- `chrome://` や `chrome-extension://` などの内部ページはPDF化できません
- ログイン認証が必要なページは、ログイン済みの状態で変換してください
- タブ数が多い場合は「タブ間の待機」を長めに設定することを推奨します
- Chromeのデバッガーを使用するため、変換中はデバッガーバーが表示されます（正常動作です）

## ファイル構成

```
tabs-to-pdf/
├── manifest.json     # 拡張機能設定
├── popup.html        # UIポップアップ
├── popup.css         # スタイル
├── popup.js          # UIロジック
├── background.js     # PDF変換処理（Service Worker）
└── icons/            # アイコン
    ├── icon16.png
    ├── icon48.png
    └── icon128.png
```
